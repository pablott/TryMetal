# Example App

**This folder is not covered during the workshop.** It exists purely as a reference for attendees to explore on their own time after the workshop. It uses the same examples we will cover but in a real production environment.

The app showcases four polished Metal shader effects integrated into SwiftUI, each in its own tab:

- **Procedural** -- Interactive dotted background with glow, repulsion, and attraction modes
- **Color** -- Image filters (grayscale, sepia, hue shift) controlled by sliders
- **Layer** -- Refracting glass lens with magnification, chromatic aberration, and Fresnel highlights
- **Distortion** -- Jelly wobble with damped spring physics and circular reveal transition

# Example App（サンプルアプリ）

**このフォルダはワークショップでは扱いません。** ワークショップ後に参加者が自分のペースで探索するための参考資料として用意されています。ワークショップで取り上げるのと同じ例を、実際のプロダクション環境で実装したものです。

このアプリはSwiftUIに統合された4つの完成されたMetalシェーダーエフェクトを、各タブで紹介しています：

- **Procedural（プロシージャル）** -- グロー・反発・引力モードを持つインタラクティブなドットグリッド背景
- **Color（カラー）** -- スライダーで操作するイメージフィルター（グレースケール・セピア・色相シフト）
- **Layer（レイヤー）** -- 拡大・色収差・フレネルハイライトを備えた屈折ガラスレンズ
- **Distortion（ディストーション）** -- 減衰バネ物理演算と円形リビールトランジションによるゼリー揺れエフェクト
